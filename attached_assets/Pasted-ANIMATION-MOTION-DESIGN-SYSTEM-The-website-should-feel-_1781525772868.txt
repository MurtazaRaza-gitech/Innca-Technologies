ANIMATION & MOTION DESIGN SYSTEM

The website should feel premium, futuristic, and highly polished.

Use animation quality similar to modern products such as Stripe, Linear, Vercel, Arc Browser, and premium fintech platforms.

The entire website should feel alive without becoming distracting.

PAGE TRANSITIONS

Implement smooth route transitions between all pages.

Requirements:

- Fade transition
- Blur transition
- Scale transition
- Smooth opacity interpolation
- Duration: 0.5s to 0.8s
- Framer Motion AnimatePresence

Page Enter:

opacity: 0 → 1
y: 20px → 0
blur: 10px → 0

Page Exit:

opacity: 1 → 0
y: 0 → -20px
blur: 0 → 10px

The transition should feel elegant and premium.

Do not use harsh movements.
